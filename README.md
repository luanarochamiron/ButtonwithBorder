# Animated Button with Border Hover Animation - CodingLab

Um componente interativo de botão moderno com animação de borda em gradiente ao passar o mouse (hover effect), desenvolvido exclusivamente com HTML5 e CSS3. O projeto explora efeitos visuais avançados sobre um fundo escuro (*Dark Mode*), ideal para dar destaque a chamadas de ação (CTA) em interfaces futuristas ou de temática gamer e tech.

## Tecnologias Utilizadas

* HTML5: Estruturação do botão e contêiner principal.
* CSS3: Estilização completa, uso de bordas arredondadas e gerenciamento de cores.
* Animações em CSS: Uso de gradientes lineares (`linear-gradient`), pseudo-elementos (`::before`/`::after`) e propriedades de transição para o efeito de brilho na borda.

## Funcionalidades e Técnicas Aplicadas

* Borda em Gradiente Neon: Aplicação de um contorno multicolorido vibrante (transicionando entre tons de amarelo, verde, azul e roxo) que envolve o botão.
* Animação Fluida no Hover: Ativação de um efeito dinâmico quando o usuário passa o mouse sobre o texto "Hover Me", destacando a iluminação e o contraste da borda.
* Design Dark Mode Premium: Botão centralizado com preenchimento cinza escuro sobre um plano de fundo puramente preto, garantindo foco total no elemento interativo.
* Centralização Absoluta: Uso de CSS Flexbox para alinhar horizontal e verticalmente o componente no centro exato da tela.
